# Flow State Game (flow-state-game)
A pipe water flow game designed to teach the Additive Solution Bias.

Created as a tool for teaching Addititive Solution Bias, the point of this game is to allow the learner to attempt to solve the puzzle with their usual tools (most likely using the additive solution bias), provide instruction on the bias, and then attempt a similiar, or more challenging puzzle, aware for the bias. The game currently locks the learner out of proceeding to level 2 without the instructor provided code. Lockout level can be set in the code, as can the unlock code.
